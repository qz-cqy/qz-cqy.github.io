题解 CF61A@洛谷 | 61A@Codeforces 【Ultra-Fast Mathematician】

发布于 2019-01-29

首先，介绍一下高大上的

异或运算符

写作'^'或"xor"

接下来，如果在最后输出bool数组，就会变成16进制，我一开始就遇到了这个QAQ、qwq。

好，紧接着，上代码：

#include <iostream>
#include <string>
using namespace std;

int main()
{
	string s1, s2;
	cin>>s1;
	cin>>s2;
	int len = s1.size();
	bool a, b;
	bool ans;
	for(int i=0;i<len;i++)
	{
		a = s1[i] - '0';
		b = s2[i] - '0';
		ans = a xor b;
		cout<<ans;
	}
	return 0;
}
